# The Opus Agent

## What You'll Learn
- Prompt structure that reduces ambiguity
- How blueprint generation helps task setup
- How to define output schemas for consistency

## Key Concepts
- Instruction clarity
- Typed output variables
- Reliability through focused tasks

## Step-by-Step Guide
1. Input raw customer feedback.
2. Agent 1: Sentiment classifier.
3. Agent 2: Product/service extractor.
4. Agent 3: Suggested support response.
5. Combine outputs into one final object.

## Challenge
Build the analyzer and submit a screenshot showing sentiment + response output.

## Tips & Best Practices
- Keep prompts explicit.
- Avoid multiple unrelated asks in one task.
