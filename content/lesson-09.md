# Sub-Workflows

## What You'll Learn
- Why modular workflows scale better
- How to define contracts between flows
- How to reuse shared automation logic

## Key Concepts
- Execute Workflow task
- Parent/child flow design
- Reusable utility workflows

## Step-by-Step Guide
1. Main workflow accepts long-form content.
2. Sub-workflow A creates social thread.
3. Sub-workflow B creates newsletter format.
4. Sub-workflow C creates short summary.
5. Return outputs to the main flow.

## Challenge
Submit screenshot of both parent and at least one sub-workflow.

## Tips & Best Practices
- Version sub-workflows intentionally.
- Keep interfaces stable.
