# Decision Agents

## What You'll Learn
- How automated branching works
- When to use human decision points
- How to structure routes for maintainability

## Key Concepts
- Decision Agent conditions
- Branch execution paths
- Priority-based routing

## Step-by-Step Guide
1. Input ticket subject and body.
2. Decision agent classifies priority.
3. High path routes to immediate review.
4. Medium path drafts reply.
5. Low path generates FAQ response.
