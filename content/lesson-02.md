# Mastering the Builder

## What You'll Learn
- How to navigate the Builder efficiently
- How to map variables across tasks
- How to debug broken links and type mismatches

## Key Concepts
- Task palette and configuration panel
- Sequential chains and data flow
- Auto-link and manual mapping

## Step-by-Step Guide
1. Add input text for a document.
2. Task 1: Extract key facts.
3. Task 2: Generate concise summary.
4. Task 3: Produce three action items.
5. Wire outputs to a single final output object.
6. Run preview and validate each task output.

## Challenge
Create a complete 3-task processor and submit a screenshot of the connected canvas.

## Tips & Best Practices
- Use predictable variable names.
- Validate each task independently before chaining.
