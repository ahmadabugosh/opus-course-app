# Integrations

## What You'll Learn
- How to configure integrations securely
- How to call external APIs
- How to combine enrichment + AI scoring

## Key Concepts
- Integration tasks
- External Service task
- Credential handling

## Step-by-Step Guide
1. Ingest lead record (name, email, company).
2. Pull additional firmographic data via API.
3. Score lead quality with an Opus Agent.
4. Draft a personalized outreach message.
5. Write back to destination system.
