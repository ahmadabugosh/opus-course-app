# Human-in-the-Loop

## What You'll Learn
- Where human review should be inserted
- How approval states affect workflow paths
- How to reduce bad automation outcomes

## Key Concepts
- Human Review task
- Agentic vs manual review
- Approval and rejection branches

## Step-by-Step Guide
1. Generate a draft post with AI.
2. Add AI checks for grammar and tone.
3. Add human review step.
4. If approved, format for publishing.
5. If rejected, return with notes.

## Challenge
Submit screenshot proving the human review stage is present and configured.

## Tips & Best Practices
- Put review near high-impact decisions.
- Keep reviewer instructions concise.
