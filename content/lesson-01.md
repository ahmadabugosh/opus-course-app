# Your First Workflow

## Before You Start

**Create your free Opus account:** Visit [opus.com](https://www.opus.com/) and sign up for a free account. You'll need this to follow along with all lessons in this course.

## What You'll Learn
- How Opus workflows are structured
- How to wire input, agent, and output tasks
- How to preview with sample data

## Key Concepts
- Workflow input and output
- Opus Agent basics
- Draft vs active workflow states

## Step-by-Step Guide
1. Create a new workflow (choose **"Build from scratch"** when prompted).
2. Add a text input variable named `raw_text`.
3. Add an Opus Agent task to summarize in 3 bullet points.
4. Connect the agent output to workflow output.
5. Run preview with sample text and review the result.

## Challenge
Build and test your summarizer workflow. Submit either:
- Workflow URL, or
- Screenshot showing successful preview output.

## Tips & Best Practices
- Name variables clearly.
- Keep one job per task.
- Test with short and long input samples.
